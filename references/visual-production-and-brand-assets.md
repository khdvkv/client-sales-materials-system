# Visual Production and Brand Assets

## Source order

1. Open the actual brand guide, vector logos, font files, current website and accepted client materials.
2. Inventory existing client photographs, illustrations, diagrams, icons and data visualisations.
3. Decide whether each asset should be retained, retouched, redrawn, regenerated or replaced.
4. Generate new imagery only for a named content role that existing assets cannot perform reliably.

## Asset register

| Field | Required value |
| --- | --- |
| Asset id and role | exact material/page and buyer purpose |
| Source and owner | client file, commissioned work, licensed source or generated output |
| Rights and privacy | allowed channel, consent, personal/medical data risk |
| Fidelity | what factual reality must remain unchanged |
| Quality | dimensions, crop, focus, noise, compression and intended display size |
| Treatment | colour, retouch, background, masking, caption and alt text |
| Approval | reviewer, date and accepted version |

## Retouching real client images

Allowed operations include crop, horizon, exposure, white balance, colour consistency, moderate noise reduction, local contrast, sharpening within the reliable resolution, dust/background cleanup and masking for layout.

Do not change medical outcome, equipment, safety condition, person identity, document content, location reality or another fact that the visual is used to prove. Material retouching requires a before/after receipt and an explicit note when the image becomes illustrative rather than documentary.

## Image2 production card

Use the image-generation or image-editing tool only after opening target images and approved references. Record:

```markdown
Material, page and buyer role:
Visual job and evidence object:
Scene, location and category reality:
People, identity, diversity, clothing and equipment constraints:

Client codes added outside the image:
Negative constraints:

Generated or edited source paths:

```

Logos, legal wording, exact numbers, product labels and readable documents are added as vector/HTML layers. Generated lettering never carries brand-critical meaning.

## Visual-richness invariant

For every section longer than five pages:

- use at least three purpose-led page modes when the content supports them: photographic proof, annotated evidence, diagram/route, comparison/table, editorial thesis, form/tool, data/result;
- prevent more than two adjacent pages from sharing the same composition grammar;
- give each page one primary information object and one clear first fixation point;
- include client-specific operating reality, evidence or material language that survives removal of the logo;
- inspect the full contact sheet for rhythm, repeated images and repeated card grammar.

This is a diagnostic floor, not a quota. A mode without a content job is decoration and fails the gate.
